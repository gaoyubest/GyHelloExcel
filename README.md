# Excel函数与公式笔记

## 公式编辑

- **只查看公式中某一部分**：选中部分公式按F9，ESC退出。
- **一次性快速选中引用部分**：选中单元格后，ctrl+[即可。
- **隐藏公式**：单击行交叉处，选中整张表->右键->设置单元格格式->保护选项卡->撤选锁定，选中隐藏->确定->工具->保护工作表，输入密码->确定->再次输入密码。
